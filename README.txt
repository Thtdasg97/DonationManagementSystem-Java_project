Project Introduction

A donation management system is a software application that helps organizations collect and manage donations. 

Functional Description

1. User Management (for Admin)
Admins can efficiently manage user accounts within the system. (Create, Read, Update, Delete)

2. Contribution Campaign Management (for Admin)
Admins have the capability to oversee and control donation campaigns.

3. Contribute to a Campaign (for User)
Users can actively participate in fundraising campaigns.

4. Manual Confirmation/Cancelation of Fund Receipt (for System Administrator)
System administrators can manually confirm or cancel the receipt of funds from donation campaigns, employing a manual verification process.

Deployment Guide (On Local)
Prerequisites
- Java Development Kit (JDK)
- Apache Maven
- MySQL Database

Steps
1. Clone the Repository
git clone https://github.com/Thtdasg97/DonationManagementSystem-Java_project
cd DonationManagementSystem-Java_project

2. Configure Database

3. Build and run
mvn clean install
mvn spring-boot:run
Access the platform at http://localhost:8080

4. Platform Access:
Open your browser and go to http://localhost:8080
